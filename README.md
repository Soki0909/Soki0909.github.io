## 🚀 クイックスタート

### 📋 **前提条件**

- Node.js 18+
- npm 9+
- Git

### ⚡ **セットアップ**

```bash
# リポジトリクローン
git clone https://github.com/Soki0909/Soki0909.github.io.git
cd Soki0909.github.io

# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev
```

### 🔧 **開発コマンド**

```bash
# 必須品質チェック（コミット前必須）
npm run format    # Prettier: 統一コードフォーマット
npm run lint      # ESLint: 厳格品質チェック
npm run build     # TypeScript: 型チェック・ビルド確認

# 開発・ビルド
npm run dev       # 開発サーバー起動
npm run build     # 本番ビルド
npm run preview   # ビルド結果プレビュー
```
